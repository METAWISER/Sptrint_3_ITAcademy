Comandos de navegación:

cd: Cambia el directorio actual.
ls o dir: Lista los archivos y carpetas en el directorio actual.
mkdir o md: Crea una nueva carpeta.
rm o del: Elimina un archivo.
rmdir o rd: Elimina una carpeta.
Comandos de administración de archivos y carpetas:

New-Item o ni: Crea un nuevo archivo o carpeta.
Copy-Item o cp: Copia un archivo o carpeta.
Move-Item o mv: Mueve un archivo o carpeta.
Rename-Item o ren: Cambia el nombre de un archivo o carpeta.
Get-Item o gi: Obtiene información sobre un archivo o carpeta.
Get-ChildItem o gci: Obtiene una lista de archivos y carpetas en una ubicación determinada.
Set-Location o cd: Cambia el directorio actual.
Comandos de administración de procesos:

Get-Process o gps: Obtiene una lista de procesos que se están ejecutando actualmente.
Stop-Process o kill: Detiene un proceso que se está ejecutando.
Start-Process: Inicia un nuevo proceso.
Wait-Process: Espera a que finalice un proceso determinado.
Comandos de administración del sistema:

Get-Service o gsv: Obtiene una lista de servicios que se están ejecutando actualmente.
Stop-Service: Detiene un servicio que se está ejecutando.
Start-Service: Inicia un servicio que se ha detenido.
Restart-Service: Reinicia un servicio.
Get-EventLog: Obtiene información de registro de eventos.
Comandos de administración de usuarios y grupos:

Get-LocalUser: Obtiene una lista de usuarios locales.
Get-LocalGroup: Obtiene una lista de grupos locales.
Add-LocalGroupMember: Agrega un usuario a un grupo local.
Remove-LocalGroupMember: Elimina un usuario de un grupo local.
Es importante tener en cuenta que esta lista no es exhaustiva y que hay muchos otros comandos disponibles en PowerShell. Además, cada comando tiene muchas opciones y parámetros diferentes que pueden personalizarse según las necesidades específicas.
